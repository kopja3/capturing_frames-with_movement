Aloita tästä

Tämä ohje on kirjoitettu aloittelijalle. Se näyttää vaihe vaiheelta, miten ohjelma asennetaan omalle koneelle ja miten selainkäyttöliittymä käynnistetään komentoriviltä.

Mitä tarvitset

  • Python 3.12
  • Git
  • internetyhteys ensimmäistä käynnistystä varten
  • webkamera tai videotiedosto

Jos Python tai Git puuttuu, asenna ne ensin. Windowsissa on helpointa asentaa ne normaaleilla asennuspaketeilla. macOS:ssä ja Linuxissa ne voi asentaa myös paketinhallinnalla.

1. Lataa projekti koneelle

Avaa komentorivi tai terminaali ja suorita:

git clone https://github.com/kopja3/capturing_frames-with_movement.git
cd capturing_frames-with_movement
git checkout master
git pull origin master

Kun tämä on valmis, olet projektin kansiossa.

2. Luo Python-ympäristö ja asenna paketit

macOS ja Linux

python3.12 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install --force-reinstall "numpy<2"
python -m pip install -r requirements-web.txt

Windows PowerShell

py -3.12 -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install --force-reinstall "numpy<2"
python -m pip install -r requirements-web.txt

Kun ympäristö on aktivoitu, komentorivillä näkyy yleensä (.venv).

3. Käynnistä Python-backendi

Pidä tämä terminaali auki ja suorita projektikansiossa:

uvicorn bird_backend_api_mask_stable:app --host 0.0.0.0 --port 8000

Kun kaikki toimii, palvelin kuuntelee portissa 8000.

4. Avaa toinen terminaali ja käynnistä selainkäyttöliittymä

Avaa toinen komentorivi samaan projektikansioon ja suorita:

python3 -m http.server 8081

Jos käytät Windowsia eikä python3 toimi, käytä:

py -3 -m http.server 8081

Tämä käynnistää paikallisen web-palvelimen. Sivu avautuu osoitteesta:

http://localhost:8081/docs/index_mask_stable_camera_select.html

5. Avaa selain myös komentoriviltä

Voit avata sivun automaattisesti suoraan komentoriviltä.

Windows PowerShell

start http://localhost:8081/docs/index_mask_stable_camera_select.html

macOS

open http://localhost:8081/docs/index_mask_stable_camera_select.html

Linux

xdg-open http://localhost:8081/docs/index_mask_stable_camera_select.html

6. Tarkista yhteys sivulla

  • Jätä backend-osoitteeksi http://127.0.0.1:8000
  • Paina Tarkista yhteys
  • Jos yhteys toimii, voit käyttää ohjelmaa
Avaa ohjelma

7. Käyttö

Videotiedosto

  • Valitse koneelta video
  • Paina Analysoi ladattu video

USB-kamera tai webkamera

  • Paina Päivitä kameraluettelo
  • Valitse haluamasi kamera listasta
  • Paina Käynnistä valittu kamera
  • Anna selaimelle lupa käyttää kameraa

Yleisimmät ongelmat

404 File not found

Et ole projektin oikeassa kansiossa tai paikallinen kopio ei ole ajan tasalla. Suorita:

git pull origin master
ls docs

Backendiin ei saada yhteyttä

Varmista, että uvicorn on käynnissä portissa 8000 ja että sivulla backend-osoite on http://127.0.0.1:8000.

Kamera ei käynnisty

Varmista, että avasit sivun osoitteesta localhost ja annoit selaimelle kameran käyttöluvan.

Portti on jo käytössä

Voit vaihtaa web-sivun portin esimerkiksi näin:

python3 -m http.server 8090

Silloin osoite on:

http://localhost:8090/docs/index_mask_stable_camera_select.html

Sulkeminen

Kun lopetat käytön, pysäytä molemmat terminaalit näppäimillä Ctrl-C.
