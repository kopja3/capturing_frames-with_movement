Linnun tunnistusohjelma

Tämä on projektin selkeä aloitussivu. Tästä pääset joko lukemaan ohjeet tai avaamaan itse ohjelman.

Valitse mitä haluat tehdä

Avaa asennus- ja käyttöohje Avaa ohjelma

Nopea käynnistys

Jos ohjelma on jo asennettu, tee näin:

uvicorn bird_backend_api_mask_stable:app --host 0.0.0.0 --port 8000

Avaa toisessa terminaalissa:

python3 -m http.server 8081

Avaa sen jälkeen selaimessa:

http://localhost:8081/docs/etusivu.html

Suora avaaminen komentoriviltä

Windows PowerShell

start http://localhost:8081/docs/etusivu.html

macOS

open http://localhost:8081/docs/etusivu.html

Linux

xdg-open http://localhost:8081/docs/etusivu.html

Huomaa: tässä vaiheessa varma aloitussivu on docs/etusivu.html. Nykyisen docs/index.html-tiedoston turvallinen ylikirjoitus ei onnistunut tämän GitHub-rajapinnan kautta.
